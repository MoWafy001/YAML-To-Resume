# How it should work

```sh
yaml-to-resume --template <template_name> yaml-file-path
yaml-to-resume --template-path <template_path> yaml-file-path
yaml-to-resume yaml-file-path
yaml-to-resume yaml-files-dir-path
yaml-to-resume --configs <configs_path> yaml-files-dir-path
yaml-to-resume yaml-file-path -o output.pdf
yaml-to-resume yaml-file-path -o resumes-dir/output.pdf
yaml-to-resume yaml-file-path -o resumes-dir/
```
